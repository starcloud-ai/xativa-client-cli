# xativa-client-cli
## Installation:
pip install --editable .
## Useage:
~~~
Usage: xativa [OPTIONS] COMMAND [ARGS]...
~~~
Options: 

   --help  Show this message and exit.

Commands:

~~~

  request-remote-data  Request remote data
  
  start-job            Start fate job.
  
  upload-local-data    upload local data

~~~
